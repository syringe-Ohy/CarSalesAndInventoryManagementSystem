# Car Sales and Inventory Management System

This project is a showroom-based application designed to streamline vehicle sales and inventory management. It provides a complete solution for managing car listings, tracking inventory, and handling customer purchases.

## 🚀 Features
- Browse available cars with detailed information
- Add, update, and delete vehicle records
- Manage inventory efficiently
- Track customer purchases and sales
- Admin panel for full system control

## 🛠 Technologies Used
- C#
- .NET Core
- SQL Server

## 🎯 Objective
The main goal of this system is to reduce manual work in car showroom management by providing an automated and user-friendly platform for handling inventory and sales operations.

## 📌 Use Case
This system can be used by small to medium-sized car dealerships to:
- Maintain vehicle stock
- Monitor sales transactions
- Improve operational efficiency

- Online payment integration
- Advanced search & filtering
